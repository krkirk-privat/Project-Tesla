What is a Tesla referral code?

A referral code, also called a referral link or discount code, is how Tesla rewards both new and existing customers. Here's the short, honest explanation.

Last updated: June 2026

When you're about to buy a new Tesla, you quickly run into terms like referral code, discount code and referral link. It can sound technical, but it covers something very simple: a personal link from an existing Tesla owner that gives you access to Tesla's current referral benefits when you order your car.

This guide explains what the code actually is, what you get out of it, how it differs from an ordinary discount, and how to use it correctly so the benefit is actually registered on your order.

Referral code, discount code or referral link?

First, a clarification, because the different words confuse many people: they mean the same thing. Tesla officially calls it a referral program. In everyday speech people often say "discount code", because the result for you resembles a discount. Whichever word you encounter, it's about the same personal link.

In this case the code is kristoffer385020, and it's built into the referral link. So you rarely need to type anything manually, you simply click the link before you order.

What do you get from a referral code?

The benefit depends on which model you buy and on the program Tesla is running at the time of ordering. Tesla adjusts the benefits regularly, so the exact amounts and kilometres should always be checked on the official referral page when you order. Recently the benefits have typically looked like this:

  • Model 3 and Model Y: a number of kilometres of free Supercharging - perfect for the first many trips.
  • Model S and Model X: a cash discount on the purchase price.

Important about the benefits

Benefits, amounts and qualifying models are set solely by Tesla and may change without notice. Treat figures here as indicative, and always confirm the current benefit on the referral page before you place your order.

How is it different from an ordinary discount code?

A classic discount code in a web shop typically deducts an amount at checkout. A Tesla referral works a little differently: it links your order to an existing owner, and the benefit is then granted according to Tesla's program. That means the benefit can be either a price adjustment (Model S/X) or a perk on top of the car, such as free Supercharging (Model 3/Y), rather than just a fixed cash amount.

It also means the code is completely free to use. You never pay more than you otherwise would at Tesla; the benefit is added on top.

How to use the code (in short)

The process itself takes under a minute:

  1. Click the referral link before you place your order.
  2. Configure and order your Tesla exactly as usual on tesla.com.
  3. The referral is registered automatically on your order, and the benefit is granted according to the current program.

Frequently asked questions
