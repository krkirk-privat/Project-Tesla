How to use a Tesla referral code

Using a referral code correctly takes under a minute, but the timing is what matters. Here's the simple step-by-step process.

Last updated: June 2026

Using a Tesla referral code is easy, but there's one thing worth knowing in advance: the referral has to be in place before you confirm your order. Do it in the right order and the benefit is registered automatically, you don't need to do anything else.

If you're new to the terms, you may want to first read what a Tesla referral code is. Otherwise, here's the process.

In short

1) Click the referral link. 2) Sign in or create a Tesla account. 3) Configure and order your car as usual. The referral comes along, and the benefit is granted according to Tesla's current program.

Step by step

  1. Open the referral link. Click tesla.com/da_dk/referral/kristoffer385020. It opens Tesla's official page with my referral attached to the session.
  2. Sign in to your Tesla account. If you don't have one, you can create it along the way. It's a good idea to be signed in so the referral is linked correctly to your profile.
  3. Select and configure your car. Choose model, colour, wheels and interior exactly as you normally would. The referral doesn't change the configuration or the displayed price.
  4. Complete the order. Pay the deposit and confirm. The referral benefit is registered on the order according to the current program.
  5. Check your account. After delivery you can see any Supercharging credits or account credit in your Tesla account / the app.

Important: use the code BEFORE you order

The most common mistake is ordering first and looking for a referral code afterwards. Tesla usually does not add a referral to an order that's already confirmed. So: open the link before you press the final order button. If in doubt, start over from the link.

What if I forgot?

If you haven't confirmed the order yet, you can often still go in via the link and order again. If the order has already been placed, contact Tesla as soon as possible and ask, but keep expectations realistic, as referrals can rarely be added retroactively.

Checklist before you press order

  • I arrived via the referral link.
  • I'm signed in to my Tesla account.
  • I'm buying a new Tesla (referrals usually don't apply to used cars).
  • I've checked the current benefit on the referral page.

Frequently asked questions

Before or after ordering? Always before. Afterwards the referral usually can't be added.

Did I forget the code? If the order isn't confirmed, try again via the link. If it's placed, contact Tesla, but don't expect a change.

App or browser? Click the link in a browser before configuring; that's the safest method.
